# Code of Conduct

Helix72 v1 is a canonical scientific artifact authored and maintained exclusively by  
**Denny Michael LaFountaine** and **Quantum Labs Research & Development LLC**.

This repository does **not** accept community participation, contributions, or collaboration through GitHub.

## Expected Behavior
- Respect the integrity of the canonical work.
- Use the repository for viewing, referencing, or academic study only.
- Do not attempt to modify, extend, or reinterpret the canon through GitHub tools.

## Unacceptable Behavior
- Submitting pull requests
- Opening issues or feature requests
- Attempting to alter canonical files
- Using the repository to promote derivative works

## Scope
This Code of Conduct applies to all interactions involving this repository.

For collaboration or licensing inquiries, contact Quantum Labs Research & Development LLC directly.
